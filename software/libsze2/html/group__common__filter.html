)

Register own filter function.

This function will be called in every szedata_read_next - if f function returns 0 - packet will be passed; if f function return other, packet will be skipped (returned NULL).

Parameters
sze
f
Returns
  • 0 - if registered correctly
  • -1 - otherwise

NOTE: hwdata_len and SZE_PACKET_HEADER_SIZE are allways alligned to 8 (see szedata packet data format). So if you expect 8B long hwdata, 4 + 8 = 12 (not aligned) + 4 = 16 (aligned), hwdata_len will be set to 12.
